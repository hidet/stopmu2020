How to use

cd build

cmake ..

make

./g4_stopmu mac/Ne1atm_gausbeam_event10k.mac

The .mac files in mac folder contain important parameters

You can specify,

 target material
 
 target density
 
 beam particle
 
 beam energy
 
 beam momentum
 
 beam direction
 
 beam position
 
 output file name
 
etc...
